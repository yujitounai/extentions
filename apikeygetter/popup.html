概要

ページ内テキスト・インライン/外部 JavaScript から API キー、トークン、 秘密情報らしき文字列を自動検出します。旧 API Key Getter と Hidden Search を統合した v1 です。

スクワッティング CDN

Polyfill.io サプライチェーン攻撃で報告された bootcdn / staticfile 等のドメインを監視します。 script・link・iframe・画像・Performance API の読み込みと HTML 内の文字列参照を確認します。

検出パターン（15種）

  • AWS アクセスキー / Credentials
  • Google API キー、Stripe、SendGrid、OpenAI
  • Slack Token、JWT、Private Key、GitHub Token など

バッジとアイコン

検出件数が拡張アイコンのバッジに表示されます。1件以上でアラートアイコンに切り替わります。 除外ドメイン（Google、Amazon 等）ではスキャンしません。

Rescan

動的に読み込まれたコンテンツを再スキャンしたい場合に使用します。

ファイル探索

現在のオリジンに対して robots.txt、sitemap.xml、.env、.git、wp-config、 actuator/heapdump、swagger、バックアップ、ページ上の script に対応する .map など、 診断で見逃しがちなパスを HEAD/GET で確認します。 200 / 301 / 302 / 401 / 403 をヒットとして表示します。

注意事項

診断許可を得たサイト、または自分が管理するサイトにのみ使用してください。 外部 script の fetch は CORS 制約により取得できない場合があります。
