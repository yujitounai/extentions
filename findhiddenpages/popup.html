概要

同一ドメイン上の隠しファイル・ディレクトリを辞書ベースで探索します。 旧 findhiddenpages と contentdiscovery を統合した v2 です。

基本的な使い方

  1. 探索したいサイトの HTTP(S) ページを開く
  2. 拡張機能アイコンから popup を開く
  3. Scan profile を選び Start をクリック
  4. ステータスが 200 OK の URL が Found URLs に表示される

Scan profile

Profile 対象 目安
Quick ルートのみ / 主要拡張子 ~3,000 URL
Standard ルート + 現在のディレクトリ ~33,000–65,000 URL
Full 全辞書・全拡張子 ~56,000–113,000 URL

初めて使う場合は Quick または Standard から始めることをおすすめします。

各ボタン

  • Start — 新規スキャンを開始（進行中のスキャンは上書き）
  • Resume — 中断したスキャンを再開
  • Stop — スキャンを一時停止（進捗は保存されます）
  • Detailed log — 404 など非 200 レスポンスもログに記録
  • Clear history — 現在ドメインの履歴を削除

popup を閉じても大丈夫？

はい。スキャンは background で動作するため、popup を閉じても継続します。 再度 popup を開くと進捗が反映されます。

注意事項

診断許可を得たサイト、または自分が管理するサイトにのみ使用してください。 大量リクエストはサーバー負荷や WAF ブロックの原因になります。
